The SYSINFO Command
===================
|version3.4|


The SYSINFO command gathers information from both the client and server side of the GNU Health HMIS node. 

The information is whown in the GNU Health client **Activity log window** (SHIFT + F12)

Some of the information retrieved by SYSINFO

**Server Side**
* Operating System
* Kernel version
* GNU HEalth server version
* Tryton kernel
* Python version
* GNU Health user

**Client side**
* GTK Client version
* Operating System version
* Python version

If you run into issues with your system, please copy the output of SYSINFO and attach it to the message. It will facilitate sorting things out.
